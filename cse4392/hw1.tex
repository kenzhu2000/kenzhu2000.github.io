\section*{Problem 3 -- 20\%}

This problem focuses on understanding different types of ambiguity in natural language, as discussed in class.

\begin{enumerate}
    \item \textbf{Types of Ambiguity}
    
    Based on the four types of ambiguity introduced in class (e.g., lexical ambiguity, syntactic ambiguity, semantic ambiguity, and discourse ambiguity), complete the following tasks:
    
    \begin{itemize}
        \item For each type of ambiguity, provide \textbf{two example sentences}.
        \item For each example, briefly explain the different possible interpretations and why it is classified as this type of ambiguity.
    \end{itemize}

    \item \textbf{Reflection Question}
    
    Answer the following questions based on your examples:
    \begin{itemize}
        \item Which type of ambiguity do you think is the most difficult for NLP systems to handle? Why?
    \end{itemize}
\end{enumerate}
